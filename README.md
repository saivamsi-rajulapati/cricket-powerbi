# 🏏 Cricket Data Analysis Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Cricket Data Analysis Dashboard** built using Power BI.
It provides insights into player performance, match statistics, and team analysis.

---

## 📊 Key Features

* Player Performance Analysis
* Match Statistics Visualization
* Team Comparison
* Interactive Filters & Slicers
* Clean and Dynamic Dashboard
